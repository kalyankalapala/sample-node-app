Goal of the project

To deploy a static webpage into the AWS EC2 instance using the CI/CD Pipelines (CodeBuild, CodeDeploy, CodePipeline) & docker containers. Also, to send an email notification to the users related to the project using the SNS topics to inform about the status of the builds and deployments in the CI/CD pipelines.

List of Tools and Services Used

  1. AWS Elastic Cloud Compute (EC2)
  2. Docker
  3. GitHub
  4. AWS CodeBuild
  5. AWS CodeDeploy
  6. AWS CodePipeline
  7. AWS Elastic Container Registry (ECR)
  8. Load Balancers & Target Groups
  9. AWS Simple Notification Service (SNS)
